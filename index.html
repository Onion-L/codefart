How

We don't touch Claude.
Claude calls us.

Claude Code has a built-in Stop hook that fires at the end of every response. We register one command. That's the whole product.

💬Youclaude "…"
→
🧠Claudethinks
→
🪝Hook firesStop event
→
💨CodeFartBRAAAP
Why

Three reasons this exists.

You walk away. We bring you back.

Claude takes 10–60 seconds. You switch tabs, check your phone, forget. The fart is your cue. It's the most effective notification you've ever ignored.

🚶‍♂️💨

Not a wrapper. A hook.

No shell aliases. No process interception. Open ~/.claude/settings.json and you'll see exactly what we added: one Stop hook entry. That's it.

🪝

Fun is a feature.

Every other notification tool is boring. This one has a theme called "wet" and a preview mode with arrow keys. You'll tell your friends about it.

Commands

One command per thing.
No config files.

Everything you'd want to do with a fart notification tool.

codefart setupWire into Claude (once)
codefart playPlay current sound
codefart listShow themes, highlight current
codefart themeInteractive picker ↑↓ Enter
codefart theme classicSwitch directly by name
codefart previewAudition without switching
codefart set-sound ~/boom.mp3Custom audio file
codefart removeClear custom sound
codefart resetBack to factory settings
codefart updateGet the latest version
codefart run -- npm run buildWrap any command 💨
