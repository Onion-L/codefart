🔊 How it works

Claude calls us.
We don't touch Claude.

No process wrapping. No shell aliases. Claude's own hook system fires codefart play when it's done. That's it.

✨ Features

Unnecessary but essential.

Every feature exists because it's funny. The utility is a bonus.

⚡

One command

curl | bash → done. Under 60 seconds from zero to first fart.

🔊

Real fart sounds

classic & wet themes built in. Add your own. Preview before committing.

🛡️

No spying

Check ~/.claude/settings.json yourself. One hook entry. Nothing else.

🕹️

Arrow key picker

↑↓ to browse themes, Enter to pick. Preview mode for try-before-you-buy.

🔄

Self-updating

codefart update. Pulls the latest. Asks for sudo only when needed.

📦

Wrap anything

codefart run -- npm run build. Fart on CI. Fart on deploy. Fart on everything.

📟 Commands

One command per thing.
No config files to touch.

Everything you'd want to do with a fart notification tool.

codefart setupWire into Claude (once)
codefart playPlay current sound
codefart listShow themes, highlight current
codefart themeInteractive picker ↑↓ Enter
codefart theme classicSwitch directly by name
codefart previewAudition without switching
codefart set-sound ~/boom.mp3Custom audio file
codefart removeClear custom sound
codefart resetBack to factory settings
codefart updateGet the latest version
codefart run -- <cmd>Wrap any command 💨
